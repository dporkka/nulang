## `counter_actor`

Counter Actor Example

Declare an actor with state and behaviors, then spawn it.
`spawn` returns an actor reference that can receive messages.

Run with: nulang examples/counter_actor.nula

### Actors

#### `Counter`

```nulang
actor Counter
```

A simple counter actor holding an integer count.
Use `inc` to increment and `get` to retrieve the current value.


## `effects`

Algebraic Effects Example

A handler intercepts a performed effect and resumes the computation
with a value. This demonstrates the core mechanism of algebraic effects:
`perform` raises an effect, `handle` catches it.

Run with: nulang examples/effects.nula


## `fibonacci`

Recursive Fibonacci with Closures

Demonstrates closures and recursion in Nulang.
Defines a recursive `fib` function as a closure bound with `let rec`,
then computes the 10th Fibonacci number.

Run with: nulang examples/fibonacci.nula

## `variant_option`

User-Defined Variant Types

Nulang has no prelude — programs declare the variants they need.
This example defines an Option-like type with two constructors,
`Some(T)` and `None`, and an `unwrap_or` helper.

Run with: nulang examples/variant_option.nula

### Functions

#### `unwrap_or`

```nulang
fn unwrap_or(o: Option[Int], default: Int) -> Int
```

Unwrap an `Option[Int]`, returning `default` when `None`.

### Types

#### `Option`

```nulang
type Option[T] = Some(T) | None
```

A simple Option type with `Some(T)` and `None` constructors.
